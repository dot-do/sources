# O*NET Occupations Data Source

Occupation reference data from the O*NET database, providing comprehensive profiles for 900+ occupations in the U.S. economy.

## Data Schema

### TSV Columns

| Column | Type | Description |
|--------|------|-------------|
| O*NET-SOC Code | string | 8-digit SOC code (XX-XXXX.XX) |
| Title | string | Official occupation title |
| Description | text | Comprehensive occupation description |

### Example Row

```
15-1252.00	Software Developers	Research, design, and develop computer and network software or specialized utility programs.
```

## Collection Mapping

Maps to: **db/Occupations/{slug}/readme.mdx**

### Slug Generation

```typescript
// Example: "15-1252.00" → "software-developers"
function generateSlug(code: string, title: string): string {
  return title
    .toLowerCase()
    .replace(/[^a-z0-9\s-]/g, '')
    .replace(/\s+/g, '-')
}
```

## Relationships

- **Belongs To**: NAICS industries
- **Related To**: Similar occupations, career paths

## Transform Logic

```typescript
interface OccupationRow {
  'O*NET-SOC Code': string
  'Title': string
  'Description': string
}

interface Occupation {
  code: string
  title: string
  description: string
  slug: string
  collection: 'Occupations'
  source: 'onet'
}

function transformOccupation(row: OccupationRow): Occupation {
  return {
    code: row['O*NET-SOC Code'],
    title: row['Title'],
    description: row['Description'],
    slug: generateSlug(row['O*NET-SOC Code'], row['Title']),
    collection: 'Occupations',
    source: 'onet',
  }
}
```

## Import Process

1. **Download**: Fetch latest TSV from O*NET
2. **Parse**: Parse TSV format
3. **Transform**: Apply mapping logic
4. **Generate**: Create MDX files in db/Occupations/
5. **Validate**: Verify all occupations imported

## Update Strategy

### Full Refresh
- Import all occupations on major version updates
- Overwrite existing MDX files
- Preserve manual edits in separate metadata

### Incremental Updates
- Track changed occupations between versions
- Only update modified entries
- Maintain revision history
