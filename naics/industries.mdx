# NAICS Industries Data Source

Complete industry classification data from the 2022 NAICS revision, covering ~1,000 industry codes across all levels of the hierarchy.

function generateIndustrySlug(code: string, title: string): string {
  return title

    .replace(/[^a-z0-9\s-]/g, '')
    .replace(/\s+/g, '-')
    .replace(/^-+|-+$/g, '') // Remove leading/trailing hyphens
}

## Transform Logic

```typescript
interface IndustryRow {
  'NAICS Code': string
  'NAICS Title': string
}

interface Industry {
  naicsCode: string
  title: string
  level: number
  parentCode: string | null
  sectorCode: string
  slug: string
  collection: 'Industries'
  source: 'naics'
}

function transformIndustry(row: IndustryRow): Industry {
  const code = row['NAICS Code']
  const title = row['NAICS Title']

  return {
    naicsCode: code,
    title: title,
    level: code.length,
    parentCode: getParentCode(code),
    sectorCode: code.substring(0, 2),
    slug: generateIndustrySlug(code, title),
    collection: 'Industries',
    source: 'naics',
  }
}

function getParentCode(code: string): string | null {
  if (code.length <= 2) return null
  return code.substring(0, code.length - 1)
}
```

## Hierarchy Building

### Top-Down Approach

```typescript
interface IndustryHierarchy {
  code: string
  title: string
  children: IndustryHierarchy[]
}

function buildHierarchy(industries: Industry[]): IndustryHierarchy[] {
  // Start with 2-digit sectors
  const sectors = industries.filter(i => i.level === 2)

  return sectors.map(sector => ({
    code: sector.naicsCode,
    title: sector.title,
    children: getChildren(sector.naicsCode, industries)
  }))
}

function getChildren(parentCode: string, industries: Industry[]): IndustryHierarchy[] {
  const children = industries.filter(i => i.parentCode === parentCode)

  return children.map(child => ({
    code: child.naicsCode,
    title: child.title,
    children: getChildren(child.naicsCode, industries)
  }))
}
```

## Description

This industry comprises establishments primarily engaged in computer software publishing or publishing and reproduction. Establishments in this industry carry out operations necessary for producing and distributing computer software, such as designing, providing documentation, assisting in installation, and providing support services to software purchasers.

These establishments may design, develop, and publish, or publish only:
- Systems software
- Applications software
- Video game software
- Software development tools
