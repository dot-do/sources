# NAICS - North American Industry Classification System

The **North American Industry Classification System (NAICS)** is the standard used by Federal statistical agencies in classifying business establishments for the purpose of collecting, analyzing, and publishing statistical data related to the U.S. business economy.

## Overview

NAICS provides a uniform framework for:
- **Industry classification** - Standardized codes for 1,000+ industries
- **Economic statistics** - Consistent measurement across sectors
- **Business analysis** - Industry comparisons and trends
- **Policy making** - Evidence-based decision support

## NAICS Structure

### Hierarchical Classification

NAICS uses a 2-6 digit hierarchical structure:

| Digits | Level | Example | Title |
|--------|-------|---------|-------|
| 2 | Sector | 51 | Information |
| 3 | Subsector | 511 | Publishing Industries |
| 4 | Industry Group | 5112 | Software Publishers |
| 5 | NAICS Industry | 51121 | Software Publishers |
| 6 | National Industry | 511210 | Software Publishers |

### Code Format

```
XX        Sector (20 sectors)
XXX       Subsector (~100 subsectors)
XXXX      Industry Group (~300 groups)
XXXXX     NAICS Industry (~700 industries)
XXXXXX    National Industry (~1000 industries)
```

## Relationships

### With O*NET Occupations

NAICS industries map to O*NET occupations through employment data:

```
Industry (NAICS 511210: Software Publishers)
  └─> Employs
      ├─> Software Developers (15-1252.00)
      ├─> Computer Systems Analysts (15-1211.00)
      └─> Computer Programmers (15-1251.00)
```

### Hierarchy

```
Sector (51: Information)
  └─> Subsector (511: Publishing Industries)
      └─> Industry Group (5112: Software Publishers)
          └─> NAICS Industry (51121: Software Publishers)
              └─> National Industry (511210: Software Publishers)
```

## Import Strategy

### Data Sources

1. **Primary**: Census Bureau NAICS files
2. **Fallback**: Local TSV file (`mdx/config/datasets/naics.tsv`)
3. **Enrichment**: Industry descriptions, employment data

### ETL Pipeline

1. **Extract**: Download NAICS code file
2. **Parse**: Parse TSV/Excel format
3. **Transform**: Generate slugs, build hierarchy
4. **Enrich**: Add descriptions, cross-references
5. **Load**: Write MDX files to db/Industries/

### Update Frequency

- **Major revisions**: Every 5 years (2022, 2027, etc.)
- **Minor updates**: Annual clarifications
- **Import strategy**: Full refresh on major versions

## Usage Examples

### Find Industry by Code

```typescript
import { db } from '@mastra/core'

const industry = await db.industries.findOne({
  where: { naicsCode: '511210' }
})
```

### Get Industry Hierarchy

```typescript
const sector = await db.industries.findOne({
  where: { naicsCode: '51' }
})

const children = await db.industries.findMany({
  where: { parentCode: '51' }
})
```

### Find Related Occupations

```typescript
const industry = await db.industries.findOne({
  where: { naicsCode: '511210' }
})

const occupations = await db.occupations.findMany({
  where: {
    industries: { has: industry.naicsCode }
  }
})
```
