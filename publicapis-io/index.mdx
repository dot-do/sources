# PublicAPIs.io

**PublicAPIs.io** is a curated, searchable directory of over 1000 free public and open REST APIs, providing a user-friendly interface for API discovery with advanced filtering and search capabilities.

## Overview

PublicAPIs.io provides:
- **1000+ APIs**: Curated collection of quality APIs
- **Live Search**: Real-time search functionality
- **Advanced Filters**: Filter by category, auth, HTTPS, CORS
- **Clean Interface**: Modern, intuitive UI
- **JSON Export**: Programmatic API access
- **No Authentication**: Open access to directory
- **Community-Driven**: Accepts contributions

## Platform Features

### Search & Discovery

1. **Real-Time Search**: Instant results as you type
2. **Multi-Field Search**: Search across name, description, category
3. **Category Browser**: Browse by organized categories
4. **Filter Options**:
   - Authentication type
   - HTTPS support
   - CORS compatibility
   - Category

### API Details

Each API listing provides:
- API name and description
- Documentation link
- Authentication requirements
- HTTPS support
- CORS status
- Category classification
- Usage examples (some APIs)

## Data Structure

### PublicAPIs.io Schema

```typescript
interface PublicAPIEntry {
  name: string                    // API name
  description: string             // API description
  url: string                     // Documentation URL
  category: string                // Primary category
  auth: AuthType                  // Authentication type
  https: boolean                  // HTTPS support
  cors: CorsStatus                // CORS support
  link: string                    // API endpoint
}

type AuthType =
  | 'None'
  | 'apiKey'
  | 'OAuth'
  | 'X-Mashape-Key'
  | 'User-Agent'

type CorsStatus =
  | 'yes'
  | 'no'
  | 'unknown'
```

## API Access

### Web Interface

Browse at: https://publicapis.io

**Features:**
- Search bar with live results
- Category navigation
- Filter by auth type
- Filter by HTTPS/CORS
- One-click documentation links

### JSON API

```bash
# Get all APIs
curl https://api.publicapis.io/api/v1/apis

# Search APIs
curl https://api.publicapis.io/api/v1/apis?search=weather

# Filter by category
curl https://api.publicapis.io/api/v1/apis?category=weather

# Filter by auth
curl https://api.publicapis.io/api/v1/apis?auth=apiKey

# Filter by HTTPS
curl https://api.publicapis.io/api/v1/apis?https=true

# Combine filters
curl https://api.publicapis.io/api/v1/apis?category=finance&auth=None&https=true
```

### Rate Limits

- **No official rate limits** on API access
- **Recommended**: Cache responses, don't spam
- **Status**: Free, community-maintained

## Integration Examples

### React Component

```typescript
import { useEffect, useState } from 'react'

function APIFinder() {
  const [apis, setApis] = useState([])

  useEffect(() => {
    fetch('https://api.publicapis.io/api/v1/apis?category=weather')
      .then(res => res.json())
      .then(data => setApis(data.apis))
  }, [])

  return (
    <ul>
      {apis.map(api => (
        <li key={api.name}>
          <a href={api.url}>{api.name}</a> - {api.description}
        </li>
      ))}
    </ul>
  )
}
```

### Node.js Script

```javascript
const fetch = require('node-fetch')

async function findAPIs(category) {
  const response = await fetch(
    `https://api.publicapis.io/api/v1/apis?category=${category}`
  )
  const data = await response.json()
  return data.apis
}

// Find all weather APIs
const weatherAPIs = await findAPIs('weather')
console.log(weatherAPIs)
```
