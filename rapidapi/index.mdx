# RapidAPI Hub

**RapidAPI Hub** is the world's largest API marketplace, connecting over 4 million developers with 40,000+ APIs through a unified platform with single authentication.

## Overview

RapidAPI provides:
- **40,000+ APIs**: Largest collection of public and private APIs
- **Unified Authentication**: Single API key for all APIs
- **API Testing**: Built-in testing console
- **Usage Analytics**: Track API consumption
- **Monetization**: API providers can monetize their APIs
- **Enterprise Plans**: Team collaboration and management
- **Multi-Protocol**: REST, GraphQL, SOAP support

## Platform Features

### For API Consumers

1. **Single API Key**: Access all APIs with one key
2. **Consistent Interface**: Unified request format
3. **Built-in Testing**: Test APIs before integration
4. **Code Snippets**: Auto-generated in 10+ languages
5. **Usage Dashboard**: Monitor API consumption

7. **Support**: Dedicated support for paid plans

### For API Providers

1. **Monetization**: Charge for API usage

5. **Management**: User management and billing
6. **Versioning**: API version control

## Data Structure

### RapidAPI Schema

```typescript
interface RapidAPIEntry {
  id: string                      // Unique API ID
  name: string                    // API name
  description: string             // API description
  category: string                // Primary category
  provider: {
    name: string
    url: string
    email?: string
  }
  pricing: {
    free: boolean                 // Has free tier
    tiers: Array<{
      name: string                // Plan name
      price: number               // Monthly price
      requests: number            // Request limit
    }>
  }
  endpoints: Array<{
    path: string
    method: string
    description: string
    parameters: object
  }>
  authentication: {
    type: 'rapidapi-key'          // Always RapidAPI key
    header: 'X-RapidAPI-Key'
  }
  protocol: 'REST' | 'GraphQL' | 'SOAP'
  baseUrl: string
  documentation: string
  rating: number                   // User rating (1-5)
  usageCount: number               // Monthly usage count
  tags: string[]
  status: 'active' | 'deprecated'
}
```

## Collections Generated

This source generates the following database collections:

### 1. RapidAPIs (~40,000 entries)

**Path**: `db/API/{category}/{provider}/{api-name}/readme.mdx`

Each API includes:
- Full API documentation
- Endpoint descriptions
- Authentication setup (RapidAPI key)
- Pricing tiers
- Code examples in 10+ languages
- Provider information
- User ratings and reviews
- Usage statistics
- Related APIs

**Example**: `db/API/Weather/OpenWeather/OpenWeather_API/readme.mdx`

### 2. API Providers (~10,000 entries)

**Path**: `db/Providers/{provider-slug}/readme.mdx`

Each provider includes:
- Provider profile
- APIs offered
- Pricing strategy
- Support contacts
- API quality metrics
- User reviews

### 3. Pricing Tiers

**Path**: `db/Pricing/{api-id}/readme.mdx`

Detailed pricing information:
- Free tier details
- Paid plan comparison
- Request limits
- Rate limiting
- Overage charges

## API Access

### RapidAPI Platform API

```bash
# Subscribe to an API (requires RapidAPI account)
# All requests use X-RapidAPI-Key header

curl -X GET "https://api.example.com/endpoint" \
  -H "X-RapidAPI-Key: YOUR_RAPIDAPI_KEY" \
  -H "X-RapidAPI-Host: api.example.com"
```

## Usage Examples

### Find AI APIs

```typescript
import { db } from '@mastra/core'

const aiAPIs = await db.apis.findMany({
  where: {
    category: 'Artificial Intelligence',
    pricing: { free: true }
  }
})
```

### Compare Pricing

```typescript
const weatherAPIs = await db.apis.findMany({
  where: {
    tags: { has: 'weather' }
  },
  include: {
    pricing: true
  }
})
```

### Find High-Rated APIs

```typescript
const topAPIs = await db.apis.findMany({
  where: {
    rating: { gte: 4.5 },
    usageCount: { gte: 1000 }
  },
  orderBy: { rating: 'desc' }
})
```
